cool1  = 酷🐟 ？
开启程序后 任务栏会出现一条小鱼。

1. 通过下拉选单选择程序
2. 点击右侧启动按钮，即可启动应用程序
3. cool1会自动记录启动历史记录
4. 按住 option 可以显示删除按钮、kill 程序按钮
5. 历史记录中的程序可以进行拖拽排序、加星、启动、删除等操作
6. cmd + Q 退出程序
